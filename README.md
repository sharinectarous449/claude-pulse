## 📱 Connecting your phone for alerts

The dashboard allows you to push notifications to your phone to approve tool calls. You must use a supported notification service to enable this. 

1. Within the application, open the Settings menu.
2. Look for the Ntfy configuration tab.
3. Generate an alert channel name in the app.
4. Download the Ntfy app on your phone.
5. Subscribe to the same channel name in the mobile app.
6. Test the connection to ensure your phone receives a signal from the software.

## 🔍 How to use the dashboard

The main screen shows your current token usage in real time. You see a live graph that updates as you write code or interact with your AI tools. 

### Managing sessions
If your chat session closes unexpectedly, the dashboard keeps a record. Click the Recovery tab to view your past sessions. Select the session you want to restore and click the Reconnect button.

### Full-text search
Use the search bar at the top of the screen to find specific moments in your history. Type a keyword or a snippet of code. The dashboard sorts your history by date and relevance.

### Approving tool calls
You can set your tools to require manual approval. When the AI attempts to run a command or use a tool, you receive a push notification on your phone. Open the alert to see the request. Tap "Approve" to let the code proceed or "Deny" to stop it.

Do I need to install Node.js?
No. The application is self-contained. You do not need to install extra software to make it run.

Is my data private?
Yes. claude-pulse operates on your local machine. Your chat history and token usage remain on your computer.

Does it support other AI coding tools?
The dashboard focuses on Claude Code workflows. It provides the best experience when you use it alongside those tools.

Can I track multiple projects?
The dashboard tracks activity based on your local workspace. You can switch between folders in the Settings menu to monitor different projects.

What if my phone stops receiving notifications?
Check your phone Wi-Fi or cellular data settings. If the connection fails, open the settings in the desktop app and click "Refresh Connection" to re-link your device.

Does this monitor my entire computer?
No. The application only tracks the activity of your coding tools as you configure them within the app. It does not watch other browser sessions or programs.
